Find log global minimizer of $f$ in $[a,b]$ for inexact composite step methods. Performs a scalar linesearch on the parameter $ x $ of $ dn+x*Dt $ such that this sum minimizes the function $ f $ It will start from the upper bound and proceed to the lower bound logarithmically, however it will stop, as soon as the norm of $ dn+x*Dt $ increases for decreasing $ x $. This is possible, if dn and Dt are not orthogonal.
