Detailed Description

template<class F, class DF, class DDF>
class Spacy::FEniCS::C2Functional< F, DF, DDF >

Functional interface for FEniCS. Models a twice differentiable functional \(f:X\rightarrow \mathbb{R}\).

Template Parameters
Fdolfin::Form describing the functional value
DFdolfin::Form describing the derivative
DDFdolfin::Form describing the second derivative
Warning
In the .ufl file you have to name the argument of \(f\) by "x"!
